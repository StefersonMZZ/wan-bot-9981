### TYPE IN TERMUX
```bash
> pkg update && pkg upgrade
> pkg install git -y

$ pkg install imagemagick
$ pkg install nodejs
> git clone https://github.com/Adhwa9981/wan-bot-9981
> cd wan-bot-9981
> npm install
```
